前言

你了解 Hexo 吗？

Hexo 是一个静态博客框架（Wordpress 是动态的），基于 Node.js，可以将 Markdown 文章通过渲染引擎，生成一个静态网页，再结合 Git 命令（ssh），Hexo 是一个快速、简洁且高效的博客框架。

Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

前几天前偶然间了解到了 Hexo 这个静态博客网站，很适合那些喜欢博客的兄弟们，最重要的是它是免费的主要是免费的主要是免费的重要是事情说三遍，零基础小白也能上手，不需要编程基础，跟着操作来即可。
